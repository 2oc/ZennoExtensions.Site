Менеджер аккаунтов

Функционал многопоточной работы со списком аккаунтов.

Метод Описание
GetInstance Возвращает экземпляр класса AccountManager.
Initialize Производит инициализацию менеджера аккаунтов.Сопоставляет существующую конфигурацию аккаунтов или создает новую для файла с аккаунтами.
GetFreeAccountWithMinExecutionsCount Возвращает свободный аккаунт с наименьшим количеством запусков.
GetAccount Возвращает аккаунт, удовлетворяющий предикату.
ReleaseAccounts Освобождает все аккаунты для которых предикат возвращает true.

Настройка браузера

Гибкая генерация свойств навигатора, UserAgent’а, отпечатков и множества других параметров.

Метод Описание
BrowserManager.Generate Генерация профиля браузера.
BrowserManager.Setup Применяет конфигурацию к ZennoPoster‘y.
BrowserManager.Save Сериализует и сохраняет объект профиля браузера.
BrowserManager.Load Загружает объект профиля браузера, сохраненный в файл.
BrowserManager.LoadFromString Загружает объект профиля браузера, сохраненный в строку.

Мультилогер

Возможность одновременного логирования в несколько источников.

Метод Описание
GetInstance Возвращает экземпляр класса MultiLogger.
MultiLog Логирование через набор логгеров, установленных через свойство LogTo.
LogToProjectMaker Вывод сообщения в лог ProjectMaker.Обертка над методами project.Send***ToLog.
LogToFile Логирование в файл.
LogByCustom Логирование через набор пользовательских логгеров, установленных в свойстве CustomLoggers.

Методы расширения локальных переменных ZennoPoster

Дополнительный функционал для работы с локальными переменными project.Variables[“name”].

Метод Описание
IsEmpty Проверяет указано ли значение переменной.
IsInt Проверяет является ли значение переменной целым числом.
IsDouble Проверяет является ли значение переменной вещественным числом.
IsBool Проверяет является ли значение переменной логическим значением.
IsDate Проверяет является ли значение переменной датой.
IsFileExists Проверяет существует ли файл по указанному пути.
IsTextFileEmpty Проверяет пуст ли текстовый файл.
IsDirectoryExists Проверяет существует ли директория по указанному пути.
IsDirectoryEmpty Проверяет есть ли файлы в директории.
IsArray Проверяет является ли строка массивом данных.
IsRange Проверяет является ли значение переменной диапазоном чисел или дат.
ToRangeOfFloatNumbers Конвертирует строковое значение переменной в диапазон вещественных чисел.
ToRangeOfNumbers Конвертирует строковое значение переменной в диапазон целых чисел.
ToRangeOfDates Конвертирует строковое значение переменной в диапазон дат.
ToBool Конвертирует строковое значение переменной в Boolean.
ToInt Конвертирует строковое значение переменной в Int32.
ToDouble Конвертирует строковое значение переменной в Double.
ToDateTime Конвертирует строковое значение переменной в DateTime.
ToArray Разбивает строковое значение переменной на массив значений.
Spintax Применяет Spintax к содержимому переменной.
ThrowIfEmpty Бросает исключение, если переменная пустая.
ThrowIfNotInt Бросает исключение, если значение переменной не является целым числом.
ThrowIfNotBool Бросает исключение, если значение переменной не является логическим значением.
ThrowIfNotDouble Бросает исключение, если значение переменной не является вещественным числом.
ThrowIfNotDate Бросает исключение, если значение переменной не является датой.
ThrowIfNotArray Бросает исключение, если значение переменной не является массивом данных.
ThrowIfFileNotExists Бросает исключение, если файл не существует по указанному пути.
ThrowIfTextFileIsEmpty Бросает исключение, если файл пуст.
ThrowIfDirectoryNotExists Бросает исключение, если директория не существует по указанному пути.
ThrowIfDirectoryIsEmpty Бросает исключение, если директория пуста.
ThrowIfNotRange Бросает исключение, если значение переменной не является диапазоном.

Методы расширения ZennoLab.CommandCenter.Tab

Дополнительный функционал для работы с табом.

Метод Описание
tab.Go Выполняет переход по указанному Url. Обертка над Navigate.
tab.Refresh Выполняет обновление текущей страницы.
tab.WaitLoading Выполняет ожидание полной прогрузки страницы.
tab.GetElementByXpath Находит элемент по XPath. Обертка над FindElementByXPath.
tab.WaitFor Выполняет ожидание, пока на странице не будет выполнено условие.

Методы расширения для ZennoLab.CommandCenter.HtmlElement

Дополнительный функционал для работы с элементами страницы.

Метод Описание
element.IsHidden Проверяет скрыт ли элемент.
element.Rise Выполняет событие для элемента.
element.ClickOn Выполняет клик по элементу.
element.AttributeSet Устанавливает значение атрибуа элемента.
element.AttributeGet Получает значение атрибуа элемента.
element.ValueSet Устанавливает значение элемента.
element.ValueGet Возвращает значение элемента. Обертка над GetValue();
element.ScrollTo Прокручивает струницу к текущему элементу. Обертка над ScrollIntoView().
element.ThrowIfNull Бросает исключение, если элемент пуст (свойство IsVoid возвращает true).
element.ThrowIfHidden Бросает исключение, если елемент скрыт.
element.ThrowIfNullOrHidden Бросает исключение, если елемент пуст (свойство IsVoid возвращает true) или скрыт.
